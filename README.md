## Sobre o Curso

A disciplina de Algoritmos e Estrutura de Dados I tem como objetivo principal proporcionar aos estudantes conhecimentos fundamentais sobre algoritmos e estruturas de dados, essenciais para a análise e manipulação de dados no contexto de negócios.

## Conteúdo do Repositório
